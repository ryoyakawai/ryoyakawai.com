:: WORKS ::

Pepper sings!

Pepper is the human shaped robot that is able to move speak, dance and so on with bunch of sensors, and is able to install custom developed application to control him/her same as smart phone.

Final goal of this project was to make Pepper to sing a song, and all of those were my role. So I developed backend API to compose and generate song with existing awesme software named "VOCALODUCER", and create all of Pepper's gesture during whole process. That process was like following(as you can see from the video above):

  1. Ask user to input lyrics by using tablet.
  2. Ask user to select melody and music genre from the list.
  3. Then ask VOCALODUCER to compose music by using parameters from user.

The challenges of this projects were following:

  • Because at this moment, Pepper was not in the market, so almost no informations about developping applications for Pepper
  • Application uses Python to develop, however to me it was the first time to use that language
  • Schedule was so tight. Duration was only around 3 weeks to complete it.
  • It was my first time to design human shaped robot's gesture.

Bunch of try and error was experienced during this development, however finally I succeeded both compete to develop within tight schedule. The completion was at the event venue about 1 hour before the begening, though.

POKET MIKU („Éù„Ç±„ÉÉ„Éà„Éª„Éü„ÇØ)

POCKET MIKU is the first gadget/musical hardware instruments in the world that can sing a song with voice of Miku Hatsune, and the gadget is be able to make sing a song by playing as same way as playing the piano.

My role in this project was to develop utility application to update lyrics of POKET MIKU from browser with using Web MIDI API.
At the moment of the project, HTML5 was in early stage, and Web MIDI API was the one of the experimental web browser's API and first API that connect directly external hardware from Web browser. So this utility web application was the first web application in the world targeting for consumer hardware products that controls external hardware with using Web MIDI API.

And because source code of this utility web application was stored in public repository at GitHub, developing application of POKET MIKU was popular among DIY software developer, such as who loves Maker Faire for example, and lots of demo videos developed by those are shared on YouTube.

It was such a remarkable moment for project member and those who joined this movement.

The challenges during this projects were following:

  • To develop with brand new and experimental API to consumer product.
  • In the company, it was the first case to publish source code of consumer products application's source code.
